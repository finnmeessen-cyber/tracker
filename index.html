Hevy auf macOS · Diese Seite hosten

Diese Seite vom iPhone aufrufen

Option 1 — GitHub Pages (empfohlen, kostenlos)

Diese HTML-Datei auf GitHub hochladen und GitHub Pages aktivieren. Dann hast du eine feste URL die auf jedem Gerät funktioniert.

1. github.com → Neues Repo erstellen
2. index.html hochladen
3. Settings → Pages → Deploy from main
4. URL: https://deinname.github.io/tracker
KostenlosÜberall erreichbar

Option 2 — Lokales Netzwerk (nur Heimnetz)

Mac und iPhone im selben WLAN. Im Terminal auf dem Mac:

cd ~/Desktop && python3 -m http.server 8080

Dann auf dem iPhone im Safari: http://[Mac-IP]:8080

Mac-IP findest du unter: Systemeinstellungen → WLAN → Details

Nur im HeimnetzKein Internet nötig

Option 3 — Datei direkt öffnen

Die Datei per AirDrop ans iPhone schicken und im Safari öffnen. Daten werden lokal gespeichert — beim Schließen bleiben sie erhalten.
